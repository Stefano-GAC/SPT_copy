# SPT_copy
## 1. Descripción

    Este proyecto consiste en una mejora de la web original:

    Original: https://saunapuertadetoledo.com/
    Versión desarrollada: optimizada visualmente -> https://stefano-gac.github.io/SPT_copy/

    Objetivo:
    Mejorar diseño y usabilidad
    Aplicar conocimientos DAW
    Crear una experiencia más atractiva
## 2. Tecnologías utilizadas
    * HTML5
    * CSS3
    *JavaScript
## 3. Estructura del proyecto
    SPT_copy/
    │
    ├── index.html
    ├── styles.css
    ├── assets/ (imágenes)
    ├── README.md
    └── .git/
## 4. Descripción de archivos
    index.html:
    - Estructura principal
    - Contenido del sitio
    styles.css:
    - Estilos visuales
    - Diseño responsive
    assets/:
    -Imágenes del sitio
## 5. Instalación
    git clone https://github.com/Stefano-GAC/SPT_copy.git
    cd SPT_copy

    Abrir:
    index.html
